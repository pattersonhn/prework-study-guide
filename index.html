HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The doctype element tells browser what document type to expect when it is opened
  • The html opening tag represents the start of our HTML document
  • HTML attributes define additional information about an element and are always defined in the beginning element, never the closing element
  • Metadata is data about a webpage that the browser uses to effectively serve our webpage to users
  • Child elements are elements that are nested inside of other elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and dwitches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
